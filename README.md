# Moco
This is a Moco-app for Android.

Push

Step 1: git add filename or foldername/ (make sure you don't forget "/")

Step 2: git commit -m"your comments"

Step 3: git push -u origin master

Delete a directory

Step 1: git rm -rf directory

Step 2: git commit -m"your comments"

Step 3: git push -u origin master
